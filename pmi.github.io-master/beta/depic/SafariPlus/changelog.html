Safari Plus Changelog

1.7~b8 (July 15, 2019)

  • New Features:
    • Show Full Site URL
    • Swipe Gestures for Toolbar
    • Custom Search Engine
    • Custom User Agent (Mobile + Desktop)
    • An option to make the scroll position of the tab manager start at the current position of the tab switcher
    • Option for large titles (on iOS 11 and above)

    Improvements:
    • Full support for 12.2
    • Reworked and simplified preferences (again)
    • Tab Bar integration for Locked Tabs (on iOS 9 and above)
    • Tab Manager now shows whether a tab is locked (if the feature is enabled)
    • Tab Manager now has an option to batch lock and unlock tabs (if the feature is enabled)
    • Tab Manager now has an option to batch-export to bookmarks
    • Improved Tab Manager reload animation
    • Fixed suppress mailto alert option for iOS 12 and above
    • Improved desktop mode switch
    • Fixed a bug that could cause a video file to be randomly deleted under certain rare circumstances when tapped (let's hope this never actually happened to anyone besides me?)

1.7~b7 (June 20, 2019)

  • New Features:
    • An additional reload button can now be added to the toolbar inside toolbar reordering options
    • Added an option to auto save finished downloads to the media library
    • Added an option to prevent downloads from being saved to the history if they're started from private browsing mode
    • Added an option to have a progress bar under the downloads button inside the toolbar (like on iOS 13), enabled by default

    Improvements:
    • Preview actions for way more files inside file browser (text files, pdf, images, etc.)
    • Better file icons
    • Better directory icon (only on iOS 12, 11 and below still use the old one)
    • Fixed some preference crashes with the toolbar reordering options on iOS 8
    • The UI for download cells has been fully rewritten, now supports dynamic type
    • Fixed a few issues that could cause downloads to become stuck (for real this time, hopefully...)
    • If a download fails due to a connection error, it now cancels and a "Download Error" notification shows (previously it would just become stuck)
    • Fixed a potential issue with video downloading

1.7~b6 (June 1, 2019)

  • New Features:
    • The ability to manually start downloads from an URL, accessible from the plus button inside the top bar of the downloads list

    Improvements:
    • Unsupported options will now automatically be hidden (based on the iOS version you're running)
    • The "copy link" action has been enabled on all downloads, previously it was only available on video downloads
    • Row animations inside the file browser and the downloads list are now way more smooth (Tab Manager soon)
    • The restart button on finished downloads has been improved to use the code related to manual downloading, it no longer opens the website and rather restarts the download directly now
    • The show button on finished downloads will now only show if the file exists
    • The show button on finished downloads has been improved to scroll to the file and highlight it inside it's directory
    • The touchable area of the suggestion insert button has been increased
    • The close button inside the Tab Manager now shows a confirmation alert
    • Fixed the suggestion insert button crashing on iOS 9 and 10
    • Fixed some crashes related to the Tab Manager
    • Fixed a potential issue that could have been related to downloads becoming stuck
    • Fixed a bug that would sometimes cause the file browser to not refresh correctly
    • Fixed a rare crash when pressing the video download button
    • Fixed a crash when multiple downloads end at the same time
    • Fixed an issue that sometimes caused the rocketbootstrap error alert to incorrectly appear
    • Internal code improvements

1.7~b5 (May 9, 2019)

  • Improvements:
    • Tab Manager will now show tab icons on iOS 12 if they are enabled inside Safari settings
    • Fix some crashes related to the Tab Manager
    • Correct preference header aspect ratio on iPads and larger devices

1.7~b4 (May 1, 2019)

  • New Features (see depiction for more details):
    • Suggestion Insert Button
    • Tab Manager (Not fully done yet)

    Improvements:
    • Support for A12 / arm64e
    • Merged fixes from 1.6.8
    • Show an when a download is attempted to be started under cellular data while the 'Only Download on Wifi' option is enabled
    • Renamed 'Second 'Open in new Tab' option to 'Both Tab Open Actions' and changed it's behaviour accordingly
    • FullSafari compatibility has been restored
    • Updated arabic localization, thanks to iA7myd

1.7~b3 (April 18, 2019)

  • Improvements:
    • Several fixes for RTL languages (related to locked tabs and toolbar reordering)
    • Fix stupid oversight that caused lock button icons to not appear on 32 bit devices
    • Minor improvements to how locked tabs are handled

1.7~b2 (April 15, 2019)

  • Improvements:
    • Fixed an issue where the downloads button would not appear when the reordering options are disabled

1.7~b1 (April 15, 2019)

  • New Features (see depiction for more details):

    • Biometric Protection
    • Custom Start Site
    • Show Tab Count
    • Toolbar Reordering Options
    • Top Bar Reading Button Color
    • Top Bar Tab Bar Close Button Color
    • Tab Switcher Toolbar Background Color

    Improvements:
    • Reworked settings, added killswitch
    • Fixed status bar notifications being too big on some devices
    • Fixed choppy animation when using the "Open in opposite mode" option

    • Switched to libCSColorPicker

    • Reworked some color settings to be less hacky
    • Many more bug fixes and improvements (more than I can remember)
