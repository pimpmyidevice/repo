powerlogHelperdFix

Description

  • On iOS 13, some tweaks that inject into powerlogHelperd put it into a crash loop. This causes battery drain and the battery info inside settings not to load. This tweak stops other tweaks from injecting into powerlogHelperd, therefore fixing the issue.

Compatibility

  • iOS 13, Substrate only

Version History

  • 1.0 - Initial Release
